Projeto desenvolvido para a disciplina de Programação e Desenvolvimento Web **Sistematização** do curso de ADS.

## 🚀 Como executar o projeto
Certifique-se de ter o Docker e o Docker Compose instalados.

1.  Clone o repositório:
    ```bash
    git clone [https://github.com/Soterop/Clinica-ads-sistematizacao.git](https://github.com/Soterop/Clinica-ads-sistematizacao.git)
    ```
2.  Entre na pasta:
    ```bash
    cd Clinica-ads-sistematizacao
    ```
3.  Suba os containers:
    ```bash
    docker-compose up --build
    ```

## 🛠 Tecnologias utilizadas
* **Python / Flask** (Backend)
* **PostgreSQL** (Banco de dados)
* **Docker** (Containerização)
* **Bootstrap 5 & JavaScript** (Frontend)

## 📌 Funcionalidades
* Agendamento de consultas.
* Persistência de dados com volumes Docker.
* Interface responsiva com relógio em tempo real.
